# Welcome to DMA Cert

This is a Jekyll site deployed to GitHub Pages using GitHub Actions.

## Getting Started

This site is built with Jekyll and automatically deployed to GitHub Pages whenever you push changes to the main branch.

### Features

- ✅ Jekyll static site generator
- ✅ GitHub Actions for automated deployment
- ✅ GitHub Pages hosting
- ✅ Markdown support
- ✅ Syntax highlighting
